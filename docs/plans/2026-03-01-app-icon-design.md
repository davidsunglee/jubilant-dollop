# App Icon Design: Centered Avian Portrait

## Overview

A clean, minimal app icon featuring the Avian (yellow bird) character centered on a sky gradient background. Static PNG asset for iOS and macOS.

## Canvas

- **Base size:** 1024x1024px (iOS universal)
- **macOS sizes:** 16, 32, 128, 256, 512 at 1x and 2x
- Generated by scaling down from the 1024px base

## Background

- Linear gradient, top to bottom
- Top: `#D4EEFF` (soft sky blue)
- Bottom: `#FDE8D0` (warm peachy horizon)
- Matches the app's neutral menu screen tint (`MenuScreenTint.neutral`)

## Bird (Avian)

- Centered in canvas, occupying ~60-65% of icon area
- Facing right, slight upward tilt (~10 degrees) — mid-flap optimism
- **Body:** Yellow circle with thin orange stroke
- **Belly:** Cream/light yellow oval overlay
- **Eye:** White circle with black pupil, positioned upper-right of body
- **Beak:** Orange triangle pointing right
- **Wing:** Gold ellipse, angled slightly up (mid-flap pose)
- **Tail:** 2 small feathered ellipses fanning left
- Clean geometric shapes matching the in-game SpriteKit rendering style
- Subtle drop shadow beneath bird (black ~5% opacity, slight y-offset)

## Intentionally Omitted

- No pipes, clouds, or ground — minimal composition
- No text or title
- No complex gradients on bird — flat colors with strokes

## Implementation Approach

- Static PNG asset added to `Assets.xcassets/AppIcon.appiconset`
- Swift script using CoreGraphics to draw the icon programmatically and export PNGs
- Ensures the icon matches the in-game character rendering exactly
